# Room-EZ: Long-term Real Estate Rental Service

## Project Overview
Room-EZ is a web application designed to streamline the process of renting real estate for long-term stays. The project encompasses a full spectrum of development tasks including project management, design, coding, and demonstration, using a range of technologies like HTML, CSS, JavaScript, PHP, and MySQL.

## Key Features
- **Personal Profile Management**: Users can create and modify their personal profiles.
- **Detailed Room Viewing**: Provides a detailed view of rooms available for rent.
- **Payment Process Interface**: A user-friendly payment process UI for easy transactions.
- **Robust Database Management**: Implemented Relational Database Management System (RDBMS) using MySQL and PHP for efficient handling of user profiles and payment data.

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Achievements
- Enhanced data integrity and processing speed by 25% through the implementation of RDBMS.

## Getting Started
1. Clone the repository to get the source code.
2. Set up a PHP and MySQL environment (like XAMPP).
3. Import the database schema available in the repository.
4. Run the application on your local server.
